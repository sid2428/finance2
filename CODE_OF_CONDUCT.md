# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, assume good faith, no harassment or personal attacks,
keep discussion technical. Compliance software attracts strong opinions about
regulation — argue the design, cite the rule, never attack the person.

Enforcement contact: **kartikgaikwad2828@gmail.com**. Reports are handled
confidentially. Consequences follow the Contributor Covenant's enforcement
guidelines (correction → warning → temporary ban → permanent ban).
